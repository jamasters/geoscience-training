This repository contains 4 Bootstrap Excercises and 4 Angular JS Exercises.

   To start excercise 1: >git checkout -f ex-1
   To start excercise 1: >git checkout -f ex-2
   To start excercise 1: >git checkout -f ex-3
   etc....

To get started with the site survey application you will need...

   # DEFAULT running instance of mongo.
   # Node.js installed and executable from the command line.

To install required node packages run...

   > npm install

To start the server run...

   > node site-survey.js

To run the end to end tests open your browser and navigate to...

   http://localhost:3000/test/e2e/runner.html
